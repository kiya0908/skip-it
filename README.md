# Doodle Baseball Website

This is a Next.js project built with the App Router, designed to host and showcase the Doodle Baseball game and provide related information.

## Technologies Used

- [Next.js](https://nextjs.org/) (App Router)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Font Awesome](https://fontawesome.com/)

## Project Structure

The project follows the Next.js App Router conventions:

- `app/`: Contains all route segments and pages.
    - `app/page.tsx`: The main landing page.
    - `app/games/[slug]/page.tsx`: Dynamic route for individual game detail pages.
    - `app/about/`, `app/contact/`, `app/privacy/`, `app/terms/`: Directories for static pages.
    - `app/components/`: Reusable React components (e.g., `TopPicksGame.tsx`).
    - `app/sitemap.ts`: Configuration for dynamic sitemap generation.
    - `app/globals.css`: Global styles, likely including Tailwind CSS imports.
- `data/games.json`: JSON file serving as the data source for games.
- `public/`: Static assets like images and favicons.

## Data Source

Game information is stored in `data/games.json`. This file is used to:
- List games on the main page.
- Generate dynamic game detail pages.
- Generate the dynamic sitemap.

Each game object in `games.json` should include properties like `slug`, `title`, `description`, `coverImage`, `playUrl`, `rating`, `isTopPick`, etc.

## Dynamic Sitemap

The project utilizes Next.js's App Router feature to generate a dynamic `sitemap.xml` based on the game data in `data/games.json`. This ensures all game detail pages are included in the sitemap for better SEO. The sitemap is generated by the `app/sitemap.ts` file.

## Website URLs and Navigation

The main website can be accessed at: `https://doodlebaseball.info/`

Key pages and their URL structures include:

- **Homepage:** `/` (`https://doodlebaseball.info/`)
- **Static Pages:** Accessed directly by their names, e.g., `/about` (`https://doodlebaseball.info/about`), `/contact` (`https://doodlebaseball.info/contact`), `/privacy` (`https://doodlebaseball.info/privacy`), `/terms` (`https://doodlebaseball.info/terms`).
- **Dynamic Game Detail Pages:** Accessed using the game's unique slug from `data/games.json`, following the pattern `/games/[slug]`, e.g., `https://doodlebaseball.info/games/doodle-baseball`.

## Getting Started

First, clone the repository and install dependencies:

```bash
git clone <repository_url> # Replace with your repository URL
cd doodlebaseball-v2
npm install # or yarn install, pnpm install, bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
